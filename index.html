Integrated Motor Controller PCB

McGill Robotics Team • 2025/2026

Objective: Consolidate four discrete motor-control PCBs into a single 3-channel 24V controller (Pitch, Roll, Grip) to optimize space and improve reliability for the Mars Rover arm.

Implementation: Designed a unified system utilizing an STM32G474RET3 MCU. The architecture features a 24V input stepped down via a Buck Converter (AOZ1280CI) and LDOs. Crucially, I implemented a USB backup power circuit to allow logic-level debugging without main battery power.

Telemetry Status Plugin

Aerial Design Team • Avionics

Objective: Create a safety-critical monitoring tool for non-avionics flight operators.

Implementation: Developed a custom Mission Planner plugin that parses raw MAVLink packets. The software monitors RSSI, packet loss, and latency in real-time, displaying visual warnings when connection quality degrades below safety thresholds.
