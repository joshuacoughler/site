Integrated Motor Controller

McGill Robotics Team • 2025/2026

This board replaces three individual steering boards with a single consolidated PCB. It interfaces with the CAN bus to control three independent motors (Pitch, Roll, and Gripper) via an STM32 microcontroller.

MCU STM32G474RET3
Motor Drivers 3x DRV8874PWPR
Power Input 24V (Main) / 5V USB (Backup)
Communication CAN Bus (MCP2561FD Transceiver)

Technical Implementation

  • Power Architecture: Input 24V is stepped down via a Buck Converter (AOZ1280CI) to 5V, then to 3.3V via LDO.
  • USB Backup Logic: The board can sustain low-voltage logic (MCU, Comms) via USB power even if the main 24V battery fails.
  • Signal Integrity: Includes Level Shifters (TXS0104EDR) to bridge 5V CAN signals to 3.3V logic.
