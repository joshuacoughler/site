Integrated Motor Controller

McGill Robotics Team • Electrical Division

This board replaces three individual steering boards with a single consolidated PCB. It interfaces with the CAN bus to control three independent motors (Pitch, Roll, and Gripper) via an STM32 microcontroller.

MCUSTM32G474RET3
Drivers3x DRV8874PWPR
Input24V (Main) / 5V USB
CommsCAN Bus (MCP2561FD)

Technical Implementation >

  • Power Architecture: 24V input stepped down via Buck Converter (AOZ1280CI) to 5V, then LDO to 3.3V logic.
  • USB Backup Logic: Isolates low-voltage logic (MCU, Comms) allowing debugging via USB power even if main 24V fails.
  • Signal Integrity: Utilizes Level Shifters (TXS0104EDR) to bridge 5V CAN signals to 3.3V MCU logic.
