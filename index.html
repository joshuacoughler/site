Integrated Motor Controller PCB
McGill Robotics Team | Electrical Division

Replaced three individual steering boards with a single consolidated PCB to control 3 independent motors (Pitch, Roll, Grip).

The design utilizes an STM32G474RET3 microcontroller interfacing via CAN bus. Key features include a 24V-to-5V Buck Converter, LDO regulation, and a critical USB-backup circuit that allows logic-level debugging even when the main 24V battery is disconnected.

TOOLS: Altium Designer • STM32CubeIDE
Telemetry Status Plugin
Aerial Design Team | Avionics Division

Developed a real-time Mission Planner plugin to parse MAVLink telemetry packets.

The system translates raw data (RSSI, packet loss, latency) into threshold-based visual warnings, ensuring flight operators can instantly assess link health without interpreting raw data streams.

TOOLS: Visual Studio Code • Mission Planner
